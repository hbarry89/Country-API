Introduction

The Country API is a simple RESTful API that allows you to fetch country data using the GET method. This API provides information about various countries. This documentation will guide you on how to use the API and fetch country data using the provided code snippet.

Base URL

The base URL for making API requests is:

https://raw.githubusercontent.com/hbarry89/Countries-API/main/country.json

Fetch Request Code

To fetch country data, you can use the following code snippet. Copy and paste it into your project and modify it based on your specific needs:

async function getCountry() {
    const response = await fetch("https://raw.githubusercontent.com/hbarry89/Countries-API/main/country.json");
    const countryData = await response.json();
    console.log(countryData);
    }

getCountry();

This code snippet defines an asynchronous function getCountry() that makes a GET request to the API's endpoint to fetch country data. The response is then converted to JSON format, and the resulting data is logged to the console. You can modify this code to suit your project requirements, such as integrating the fetched data into your application.

Endpoint

The API provides a single endpoint for fetching country data:

GET /country.json

This endpoint returns a JSON array containing information about multiple countries. Each object in the array represents a country and includes various properties.

Response Format

The API response is in JSON format and consists of an array of country objects. Each country object has the following properties:

  • id: The unique identifier for the country.
  • country: The name of the country.
  • flag: The URL of the flag image for the country.
  • map: The URL of the map image for the country.
  • capital: The capital city of the country.
  • languages: An array of language(s) spoken in the country.
  • currency: The currency used in the country.
  • callingCode: The international calling code for the country.
  • internetTLD: The top-level domain (TLD) for internet addresses associated with the country.
